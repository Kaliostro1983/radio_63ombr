# UI icons (`app/static/icons/ui/`)

Невеликі **inline**-іконки для кнопок і навігації (SVG, `stroke="currentColor"` — колір як у тексту кнопки, підтримка теми).

## Звідки брати нові

- **[Heroicons](https://heroicons.com/)** — MIT, є outline/solid; копіюй SVG і залишай `currentColor`.
- **[Lucide](https://lucide.dev/icons/)** — ISC; той самий підхід.
- **[Tabler Icons](https://tabler.io/icons)** — MIT.

Уникай важких наборів з сотнями залежностей; для однієї-двох іконок достатньо вставити готовий SVG у цю папку.
